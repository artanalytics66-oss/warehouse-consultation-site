# Инструкция по установке API на хостинг Reu.ru

## Шаг 1: Создайте таблицу в MySQL

1. Зайдите в панель управления хостингом Reu.ru
2. Откройте **phpMyAdmin**
3. Выберите базу данных `u3333975_default`
4. Перейдите на вкладку **SQL**
5. Скопируйте содержимое файла `create_table.sql` и выполните его
6. Убедитесь, что таблица `blog_articles` создана

## Шаг 2: Загрузите API-файл на хостинг

1. Подключитесь к хостингу через FTP или файловый менеджер в панели управления
2. Найдите корневую папку вашего сайта (обычно `public_html` или `www`)
3. Создайте папку `admin` (если её нет)
4. Загрузите файл `api.php` в папку `admin`
5. Путь должен быть: `/public_html/admin/api.php`

## Шаг 3: Проверьте работу API

Откройте в браузере: `http://ваш-домен.ru/admin/api.php`

Вы должны увидеть JSON с массивом статей (может быть пустым или с тестовой статьей):
```json
[{"id": 1, "title": "...", "category": "...", ...}]
```

## Шаг 4: Запомните URL API

Ваш API будет доступен по адресу:
```
http://ваш-домен.ru/admin/api.php
```

Этот URL нужно будет указать в админ-панели на следующем шаге.

## Безопасность

- Файл `api.php` защищён проверкой заголовка `X-Admin-Auth`
- Только GET-запросы доступны без авторизации
- POST/PUT/DELETE требуют правильный пароль
- Не публикуйте логин/пароль от админ-панели
